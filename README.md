Project : Space Shooter Game 

Project Idea :
The Space Shooter Game is a 3D interactive shooting game where the player controls a spaceship at the bottom of the screen and shoots bullets to destroy incoming enemies from the top.

The game will include:
Player movement
Enemy spawning
Bullet shooting
Collision detection
Score tracking
Increasing difficulty
Pause / Resume
Cheat mode
Bonus life
Stealth Mode
Ultra kill ability
Boss mode

Group Members Contribution 
Member 1 : Player Control System
Features:
Spaceship movement using arrow keys
Boundary restriction
Bullet shooting using mouse click
Member 2 :Enemy System
Features:
Random enemy spawning
Enemy downward movement
Difficulty increase over time
Member 3 : Game Logic System
Features:
Collision detection (bullet vs enemy)
Score system
Game Over and Restart system

Game Controls :

LEFT ARROW : Move spaceship left
RIGHT ARROW : Move spaceship right
LEFT MOUSE : Shoot bullet
P : Pause / Resume
R : Restart game
C : Toggle Cheat Mode
ESC : Exit game
X : Stealth mode
B : Big Missile Shooting



Game Flow :
Start Game
     ↓
Spawn Enemy
     ↓
Player Moves
     ↓
Player Shoots Bullet
     ↓
Check Collision
     ↓
If Hit → Increase Score
     ↓
Spawn New Enemy
     ↓
If Enemy Reaches Bottom
     ↓
Game Over


Game Rules :
Player starts with 7 lives
Score increases when enemy is destroyed
Enemy speed increases over time
Game ends when, lives = 0
Bonus life after 10 kills
